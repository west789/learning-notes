> 常见位运算问题

1. 位操作实现乘除法

   1 >> 2相当于1/2的0次方, 200>>5相当于200/2的5次方

2. 位操作交换两数

```go
func swap(a, b int) (a, b){	
	a ^= b
    b ^= a
    a ^= b
}
```

3. 位操作判断奇偶数

    ```go
   if a & 1 == 0{
       fmt.Println("偶数")
   }
    ```

4. 位操作交换符号

    ```go
   func reversal(a int) int{
   	return ~a+1
   }
    ```
